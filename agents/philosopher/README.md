# The Philosopher Agent

A wise and thoughtful AI agent designed to explore deep questions, ethical dilemmas, and philosophical reasoning.

## Capabilities

- **Philosophical Reasoning**: Provides insights drawing from the history of philosophy.
- **Ethical Analysis**: Helps analyze situations from various ethical frameworks.
- **Wisdom & Depth**: Focuses on nuanced, meaningful responses rather than simple facts.

## Configuration

This agent uses the `nvidia/nemotron-3-nano-30b-a3b:free` model (via OpenRouter) by default for high-quality reasoning.

## Usage

This agent is part of the Shinox multi-agent mesh and can be invoked via the Director or direct mentions.
